# Customer Shopping Behavior Analysis

## Overview

This project focuses on analyzing customer shopping behavior using data analytics techniques. The workflow includes data cleaning, exploratory data analysis (EDA), SQL-based business analysis, dashboard creation in Power BI, and report generation. The main goal of the project is to uncover customer purchasing trends, spending patterns, discount behavior, and subscription insights to support data-driven decision-making.

---

## Dataset

The dataset contains customer shopping information with multiple attributes related to demographics, purchasing behavior, and transaction details.

### Dataset Summary

* Rows: 3900
* Columns: 18
* Format: CSV

### Key Features

* Customer Demographics: Age, Gender, Location
* Product Details: Category, Item Purchased, Color, Size
* Transaction Information: Purchase Amount, Payment Method, Frequency of Purchases
* Discounts & Promotions: Discount Applied, Promo Code Used
* Customer Insights: Subscription Status, Review Rating

## Tools & Technologies

## Project Workflow

### 1. Data Loading

* Imported dataset into Python using Pandas
* Checked dataset structure and missing values
* Verified data types and column consistency

### 2. Data Cleaning

* Removed duplicate records
* Handled missing or inconsistent values
* Standardized categorical values
* Prepared clean data for analysis

### 3. Exploratory Data Analysis (EDA)

Performed detailed analysis to identify:

* Customer spending behavior
* Most purchased product categories
* Impact of discounts on sales
* Subscription vs non-subscription spending
* Seasonal purchasing trends
* Customer review patterns

### 4. SQL Analysis

Executed SQL queries to derive business insights such as:

* Top-selling products
* Average customer spending
* Revenue contribution by category
* Purchase trends by subscription status
* Discount and promo code analysis

### 5. Dashboard Creation

Built an interactive Power BI dashboard containing:

* KPI Cards
* Bar Charts
* Pie Charts
* Trend Analysis Visuals
* Customer Segmentation Insights
* Revenue & Spending Comparisons

### 6. Reporting & Presentation

* Prepared project report summarizing findings
* Highlighted business insights and recommendations

## Key Insights

* Subscribers generally contributed higher overall revenue.
* Discounts significantly influenced purchasing behavior.
* Certain product categories consistently generated maximum sales.
* Seasonal and promotional trends impacted customer spending.
* Customer reviews helped identify popular products and buying patterns.
